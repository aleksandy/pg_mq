# pg_mq

> A simple asynchronous message queue for PostgreSQL


## Features

* No polling!
  * Messages are pushed to listening consumers
  

## Installation

1. Create a database
2. Execute [./schema.sql]
3. Create a listener. See [./listen.sql] and [./consumer.py] for examples.
4. ...
